Open This First

Game-Changrs Restore Pack

If a new person is restoring Game-Changrs on a laptop, this is the first file to open. It tells them which file to read next, where code comes from, where env files come from, and which supporting docs explain local ops and platform setup.

Use This Order

1

Open the main restore guide.

LATEST_RESTORE_GUIDE.html
2

If using Codex to do the restore, open the Codex prompt.

CODEX_CLEAN_SLATE_RESTORE_PROMPT_CURRENT.txt
3

If local operator work is needed after restore, open the local-ops guide.

LOCAL_OPS_SERIES_MANAGEMENT_START_HERE_CURRENT.md
4

Use the simplified system map for context.

GAMECHANGRS_SYSTEM_MAP_CURRENT.html
5

Use the detailed platform inventory only when exact platform/config ownership is needed.

GAMECHANGRS_PLATFORM_INVENTORY_DETAIL_CURRENT.html

Very Important Distinction

  • The repo code comes from GitHub or from the git bundle in the restore-point backup.
  • The env files do not come from Git.
  • The env files come from the OneDrive restore-point backup.
